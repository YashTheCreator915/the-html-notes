ID AND CLASSES

In HTML id and classes are the special attributes that can be used with any tag.

They are attributes that require a value and has a specific purpose in HTML. Lets talk about purpose of each of them separately.

#ID: ID are used to give a unique name to a tag that name can't be used for any other tag in the code. It means we can't give ID we have given to a tag to          any other tag in code and also only one id can be given to a tag.
         One more intresting feature about id is that if you want to give a link to user t another part in site then it is given as href=#id of part. You will learn          about this later on in linking section.

#CLASS: Class on other hand can be given to multiple tags as it is used to refer to a group of tags used for a specific purpose. In simple words you can give a class to multiple tags and multiple class to a tag.

*Remember you can give two or more class to tag by just adding space like class="class1 class2" where class1 and class2 are two different tag, while you can't do this with id as id is unique and each tag can have only one id.

SEMANTIC TAGS

In HTML tags which define the user something about the content are called SEMANTIC TAGS. This tags are used to make the code appear in more systematic manner

Semantic tag not only tell people about the code but also help browser to understand it and for same reason you guys must know about this tags. So here is a clear list of semantic tags.

  • <header> is used to define heading of document.
  • <footer> is used to define ending of document.
  • <main> is used to define the main content of a document.
  • <article> is defines a self-contained, independent piece of content like a blog post, news article, or comment.
  • <aside> is used to show sidebar in document.
  • <figure> is used for images used in document.
  • <figcaption> is used to define the information about image
  • <section> is used to define different parts of document with a heading and its explanation.
  • <nav> is used to define navigation bar.
  • <time> is used to defines a date/time.
  • <details>is used to defines additional details that the user can view or hide.
  • <mark>is used to defines marked/highlighted text.
  • <summary>is used to defines a visible heading for a <details> element.

LINKS

Linking just mean to link different pages or site or different section of same site together.

We use anchor tag (<a>) for linking purpose. Some of important information about it are:-

It is a container tag and text appear between its starting and ending tag is seen as link.

ATTRIBUTES
# href it is main attribute of anchor tag that contain the url of part we wanna to link.
# target this attribute is use to show where the url or linked document will be displayed when link is clicked.
Some of its value are:-

  • self: Opens link in the same current tab (default).
  • blank: Opens link in a new tab or window.
  • parent: Opens link in the immediate parent frame.
  • top: Opens link in the absolute full window, overriding frames.
  • framename: Opens link in a designated, named frame.

In HTML we can link any sites or section of same site together in two ways

INTERNAL LINKING internal linking refers to linking two or more section of same site or diferent pages of same site. This is also called relative linking. (no need tto specify target)

There are two ways in internal linking. This are:-
Two or more section of same site- if you look at side bar of this site you will notice that when you click at any of the button for a section, that section appear in front of you. To do this you just have to mention (href="#idofsection").
Two or more page of a site- to add multiple page in a site you just need to mention it's name in href if in same folder (recommended)or if the page lie in a different folder then you need to just writes its url in href.

EXAMPLE

INPUT

   <a href="#id">IMAGES (SECTION OF SAME SITE) </a>
   <a href="name.html">Different pages of same site(here I don't have taken any other page so link wouldn't work)</a>
OUTPUT IMAGES (SECTION OF SAME SITE)
Different pages of same site(here I don't have taken any other site)

EXTERNAL LINKING external linking refers to linking two or more different sites. It is also called absolute linking. You just need to specify the original url of site you are connecting in href.

INPUT

   <a href="https://www.google.com" target="_blank">GOOGLE</a>
OUTPUT GOOGLE

  • poster this tag is use to add a poster to video if we want to.
