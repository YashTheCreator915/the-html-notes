Web development refers to building and maintaining a website .Web developers use language like HTML , CSS , JAVA SCRIPT to creat there website.All these languages are widely use with each other to creat a better website.

WHAT IS HTML?

HTML stands for Hyper Text Markup Language which is generally use to creat basic structure of the web pages.

In all the website you see like of amazon, flipkart all are made using the html as its base. Still we have other languages like CSS and JAVA SCRIPT to make the site more attractive and accessible as html does't give much scope for this purpose

The people commonly take it as a programming language which is wrong. This is because the HTML it lacks fundamental programming feature such as logic,calculation and algorithms , unlike other language like PYTHON and JAVA with many loops, variable and calculations.

Parts of HTML

The HTML normally consist of three parts or segments. This are

#TAGS are the coded html command that indicates how parts of web pages should be displayed.Ex- <head>

#ELEMENTS are part of html code that represent one complete individual componet. EX- <head>and </head>

#ATTRIBUTES are special word used alond with tag to give additional informattion about the tag. EX- src, href, etc.


This are some information you must know about HTML. Now befor we actually code lets give you some information you must know about all of this:-

#Tags have starting as well as ending tag are called container tag while tags with onluy starting tag are called empty tag.

#HTML is not case sensitive which means we can use both capital as well as small letters with it.

#There have been several version of html and the latest one is HTML5.

#HTML can be coded in notepad as well but it is recommended to code in text editor like VScode for better experiance.

#HTML doesn't take any extra space or line break unless special tags are used.

#HTML file is save as name.html or name.htm.

BASIC FORMAT OF HTML

   <!doctype>
      <html lang="en">
        <head>
            <title>...</title> 
         </head>
         <body>
              ...
         </body>
      </html>

Here you can see various tags used. This all tag are important part of every HTml document and without them any HTML document can't be created.


Let us see about each one of them :-

# <!DOCTYPE> -

  • This tag is not a valid HTML.
  • It is placed at top of HTML document.
  • It tells the browser about the version of HTML use for making the document.

# <HTML> -

  • It tells the browser that the language use to make the document is HTML.
  • TYPE:- Container tag.
  • ATTRIBUTE:- Lang attribute is used along with this tag.
    •                            It tells the browser about language use in HTML document.
                               Some of its values are: en(english)(default), en-us(us english), en-uk(uk english), ar(arabic), hi(hindi), fr(french).

# <HEAD> -

  • It is used to give document information.
  • TYPE:- Container tag.

# <TITLE> -

  • It defines the document tile.
  • It is used inside <head>.
  • TYPE:- Container tag.

># <BODY> -

  • It contain all the content of html document.
  • TYPE:- Container tag.
  • ATTRIBUTE:- BACKGROUND- It is used to make a image as website background.
                               BGCOLOR- It is used to add color at background. It takes url of image.
                               LINKS- It is used to change color of all the links in the document. It takes color name as value.
                               VLINK- It is used to change the color of visited links. It takes color name as value.
                               ALINK- It is used to change the color of active links. It takes color name as value.
                               TEXT- It is used to change the color of all text in document.It takes color name as value.
                               TOPMARGIN- It is used to add some margin from top. It takes pixels as value.
                               LEFTMARGIN- It is used to add some margin from left. It takes pixels as value.
    In present use of many of this attributes is not recommended as now we use CSS for this purpose.
    If we define different color or margin to other tag inside the body tag, those tags will folllow the defined style.

By seeing about the attribute you must be thinking how to use them. You must see at HTML tag of example code I have give earlier it contain lang="en". This is a attribute which have a value. There are certain attributes which don't contain a value and are just written without any = but if a attribute has a value and if the value is of more then 1 word then it must be enclose between "and".

HEADINGS TAG

In html there are 6 heading tags that range from h1 to h6 in which h1 is largest heading while h6 is smallest. The deffault size of text is h3.

They are container tag.

They only have align attribute which take value as left, right or center.

Let's see by example

INPUT
